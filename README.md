#Pandas Data Analysis & Feature Extraction

Welcome to this repository! This project contains a collection of Jupyter Notebooks dedicated to learning and practicing core data manipulation, analysis, and feature extraction techniques using Python's Pandas library. 

The practical examples and exercises in these notebooks are built around an included Anime dataset.

##Repository Structure

Here is a breakdown of the notebooks included in this repository and what they cover:

* **`Series.ipynb`**: Introduction to Pandas Series, creating and manipulating 1D data structures.
* **`DataFrames.ipynb`**: Core concepts of working with 2D tabular data, including indexing, selection, and basic DataFrame operations.
* **`MissingData.ipynb`**: Techniques for identifying and handling null values or missing data points (e.g., `dropna`, `fillna`).
* **`GroupByAggregation.ipynb`**: Grouping data based on categories and applying aggregate functions like mean, sum, and count to summarize data.
* **`Merging_Joining_Concatination.ipynb`**: Methods for combining multiple DataFrames using merge, join, and concatenate operations.
* **`Operations.ipynb`**: Common Pandas operations, including applying custom functions, sorting, and filtering data.
* **`PivotTables.ipynb`**: Creating spreadsheet-style pivot tables for advanced data summarization and multi-level indexing.
* **`FeatureExtraction.ipynb`**: Practical techniques for extracting and engineering useful features from raw data to prepare it for machine learning models or deeper analysis.

##Dataset

* **`anime.csv`**: The primary dataset used throughout the notebooks to demonstrate Pandas operations on real-world tabular data. 

##  Getting Started
To run these notebooks locally, you will need Python installed along with Jupyter and Pandas.
